How to Set Up Hexo and Deploy to Github Page (right after setting up and deploying)

Posted on 2020-07-01

Set up environment

  • Install git (really? https://git-scm.com/)
  • Install Node.js (From official website: https://nodejs.org/en/download/)

Installing Hexo

Install Hexo command line

Initialize Hexo

Notes:

  • Configuration file for
    • Hexo project: <hexo_root_path>/_config.yml
    • Hexo theme: <hexo_root_path>/themes/<theme_name>/_config.yml
  • Create draft folder is not exist:
    • $ mkdir ./source_drafts

Start local server

Go to http://localhost:4000 and here is the hello world.

Deploy to Github Page

Set up Github Page

  • Register Github account
  • Follow instructions on https://pages.github.com/ and create site username.github.io

Set up deloyment tool

Install deployment plugin

Set up SSH key

  • Generate SSH key. Follow Github’s detailed guid or
  • Add public key (example file: C:\Users\Administrator\.ssh\id_rsa.pub) to Github on New SSH Keys
  • Verify
  • Update the deployment section Hexo config file

Install and use theme

  • Download / clone the theme
  • Copy and paste the theme folder to <hexo_root_path>/themes/<theme_name>
  • Update <hexo_root_path>/_config.yml

Random Notes

  • Generate “Read More” for posts (for Next theme, original discussion)

    • Add <!--more--> in post markdown file. Only contents before the tag would appear on home page.
    • Add scroll_to_more:false to theme config file, to allow “Read More” redirect to start from the top of the post
